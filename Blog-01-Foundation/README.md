# Blog 1: Foundation - Basic LED Blink

**"ESP32 Adventures: Your First Steps into the IoT Universe!"**

## What This Does
- Blinks the built-in LED on Pin 2
- Sends serial messages for debugging
- Foundation for the entire series

## Hardware Required
- NodeMCU-32S ESP32 board
- USB cable

## Quick Start
```bash
cd Blog-01-Foundation
pio run --target upload
pio device monitor
```

## Learning Objectives
✅ PlatformIO setup  
✅ Basic GPIO control  
✅ Serial communication  
✅ Firmware flashing
